# brownian_motion

Repo para el trabajo de la parte de fluidos de la asignatura de Bioísica del grado de Biotecnología en la UPM.

La proopuesta es **estudiar el movimiento browniano desde cero**. Para ello se nos ocurre que podriamos seguir los siguientes pasos: 

1. **Realizar el experimento** En una primera fase hemos buscado artículos donde hayan realizado el experimento y estamos recopilando información para grabar un movimiento browniano tenemos.
2. **Tracking de las partículas** Utilizando OpenCV, o cualquier otra librería de Python, queremos hacer un programa que siga a las partículas y que guarde las posiciones. Además, este programa nos podría servir para ver cómo se mueven las bacterias.
3. **Analizar los resultados** del movimiento de las partículas.
4. **Simular el movimiento browniano** y algún otro tipo de movimiento como Levy flights en python.

Todo el trabajo tiene que estar bien documentado para que los compañeros del próximo curso puedan usarlo, o seguir avanzando en el trabajo.

Este trabajo, si sale bien, podamos publicarlo en alguna revista: Revista española de física, American journal of physics or Biophysics.
